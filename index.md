## Projects

### 专病医疗大模型与多 Agent 智能诊疗系统

- 基于 Qwen3 构建 Label-guided CoT 医学推理数据
- 使用 Qwen3-14B 进行全参数监督微调
- 基于 LangGraph 设计多 Agent 诊疗流程
- 融合 RAG、患者记忆和安全规则，支持诊疗建议生成与风险校验

### TableClaw 表格智能体

- 面向复杂 Excel 和企业表格分析场景
- 构建 Agent Loop、表格检索、Schema 解析和确定性表格工具
- 通过 Domain Pack 管理业务口径、指标别名和领域知识
- 使用 Trace 和 Harness 做评测、错误归因与回归验证

### DrugRepSS 药物重定位模型

- 融合大模型语义表示与图神经网络

- 使用图对比学习和多任务损失进行药物重定位预测

### 肺炎生存风险动态预测

- 整合患者临床异构数据

- 使用多头自注意力模型识别高风险患者
